# Formeln / Zusammenhänge

>[!NOTE]
>Für alle Schaltungen gilt:
>$Pg = P1 + P2 + P3 + ... Pn$

## Reihenschaltung

- $Rg = R1 + R2 + R3 + ... Rn$

- $Ug = U1 + U2 + U3 + ... Un$
- $Ig = I1 = I2 = I3 = ... In$

## Parallelschaltung

- $1/Rg = 1/R1 + 1/R2 + 1/R3 + ... 1/Rn$  
  ODER: 
  $Rg = 1 / (1/R1 + 1/R2 + 1/R3 + ... 1/Rn)$

- $Ug = U1 = U2 = U3 = ... Un$
- $Ig = I1 + I2 + I3 + ... In$
