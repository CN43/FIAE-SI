# Elektrische Ladung 

Die **elektrische Ladung** bestimmt, wie groß der Elektronenüberschuss oder Elektronenmangel eines Körpers ist. Sie beschreibt also die elektrische Eigenschaft der Atome.


| Formelzeichen | $Q$           |                                               |
| ------------- | ------------- | --------------------------------------------- |
| Einheit       | $C$ oder $As$ | *Colomb oder Amper-Sekunde*                   |
| Formel        | $Q = I * t$   | $I$ = [[#Elektrische Stromstärke]] $t$ = Zeit |

# Elektrische Stromstärke

Die elektrische Stromstäke (electric current intensity) gibt an, wie viele elektrische Ladungen in einer bestimmten Zeit durch einen Leiter bewegt werden.

| Formelzeichen | $I$         |                                                                  |
| ------------- | ----------- | ---------------------------------------------------------------- |
| Einheit       | $A$         | *Amper*                                                          |
| Formel        | $I = Q / t$ | $Q$ = [[#Elektrische Ladung]]  $t$ = Zeit                        |
|               | $I = U / R$ | $U$ = [Spannung](#Elektrische Spannung) $R$ = elektr. Widerstand |

# Elektrische Spannung

Die elektrische Spannung gibt an, wie viel Arbeit bzw. Energie nötig ist, um ein Objekt mit einer bestimmten elektrischen Ladung innerhalb eines elektrischen Feldes zu bewegen. Es ist der Potenzialunterschied zwischen zwei Polen  je höher der Unterschied, umso höher die Spannung. Der angestrebte Ladungsausgleich ist die Ursache für einen Stromfluss.

| Formelzeichen | $U$         |                                                             |
| ------------- | ----------- | ----------------------------------------------------------- |
| Einheit       | $V$         | *Amper*                                                     |
| Formel        | $U = R / I$ | $I$ = [[#Elektrische Stromstärke]] $R$ = elektr. Widerstand |
